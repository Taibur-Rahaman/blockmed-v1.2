## 🚀 Quick Start (3 minutes)

### 1. Understand the Roles
See table above or [QUICK_START_PERMISSIONS.md](./QUICK_START_PERMISSIONS.md)

### 2. Check Your Role
```javascript
const { role } = useStore()
console.log(role) // 1, 2, 3, 4, 5, or 6
```

### 3. Use Helper Functions
```javascript
import { canAccessFeature } from '../utils/helpers'
canAccessFeature(role, 'canDispense') // true/false
```

### 4. Protect Pages
```jsx
import { FeatureAccess } from '../components/RoleProtectedComponent'
<FeatureAccess feature="canDispense" allowedRoles={[1, 3]}>
  {/* Your page */}
</FeatureAccess>
```
